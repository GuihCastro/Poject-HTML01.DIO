História dos computadores

Os computadores surgiram basicamente como enormes máquinas de cálculo, com a função de realizar muitos cálculos complexos em pouco tempo, no contexto da segunda guerra mundial.
Tinham um tamanho muito grande, ocupando salas inteiras, e não eram nada portáteis. <-

“Meus filhos terão computadores sim, mas antes terão livros.” - Bill Gates -

Relação Client x Server

Client, ou cliente, é a máquina do usuário, que fará a parte mais “ativa” da navegação, enviando requisições HTTP para o servidor e recebendo diversas informações deste.
Servers, ou servidores, são as máquinas responsáveis por armazenar todos os dados, arquivos e demais informações dos sites e aplicações web, enviando-os para o client em resposta às requisições HTTP que recebe dele. <-

Estrutura básica de um HTML

A estrutura básica de um HTML é composta por:

  1. Head

    2. Inclui o cabeçalho da página e todas as demais informações que serão carregadas antes de exibir o corpo do arquivo para o usuário

  2. Body

    3. É o corpo do arquivo HTML, que carrega todo o conteúdo que será exibido e utilizado pelo cliente. <-

Tags

O HTML não é uma linguagem de programação, mas sim de marcação de hipertexto que roda do lado do cliente (client-side).
As tags são os marcadores que ele utiliza para se comunicar com o browser e estruturar a página. <-

Diferença entre listas ordenadas e não ordenadas

  1. Listas ordenadas enumeram seus itens, de 1 em diante;
  • Enquanto as listas não ordenadas apenas os organizam através de marcadores como pontos, setas, etc.
